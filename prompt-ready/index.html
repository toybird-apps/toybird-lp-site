FAQ

よくある質問

Prompt Readyは、生成AIそのものではなく、生成AIへ渡す前の依頼文を整理するためのアプリです

Prompt ReadyだけでAIの回答を得られますか？

いいえ
Prompt Readyで依頼文を作成し、内容を確認してからChatGPT、Gemini、Claudeなどへ貼り付けて利用します

Windows版とmacOS版で機能は違いますか？

主要な依頼文作成機能、活用アイデア、最近の依頼文の復元など、基本機能は共通です
OSごとの操作体系に合わせて画面や起動方法の一部が異なります

生成AIの有料プランは必要ですか？

Prompt Readyは特定の生成AIプランを必要としません
作成した依頼文は、利用している生成AIサービスの利用条件に従って使用します

入力した文章は保存されますか？

最近コピーした依頼文は最大20件まで端末内に保存され、個別または一括で削除できます
外部サーバーへ自動送信されることはありません
