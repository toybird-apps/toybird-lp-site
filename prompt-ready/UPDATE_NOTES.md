# Prompt Ready LP 制作メモ

## 公開URL

`https://lp.toybird.com/prompt-ready/`

## キーメッセージ

- メイン：生成AI活用を、日々の業務から底上げする
- サブ：いつも同じ依頼文から抜け出し、使い方の幅を広げ、意図に近い回答へ
- ポジショニング：生成AIの知識を学ぶための教材ではなく、日々の業務で実際に使うための実践的ツール

## 主な訴求

- いつも同じ依頼文になる
- どういう使い方があるか分からない
- 意図した回答が得られない
- 用途、読み手、雰囲気、出力形式などを整理して依頼文を作成
- 活用アイデアから見落としがちな使い方を発見
- ChatGPT、Gemini、Claude、その他の生成AIに対応
- 入力内容は端末内で処理し、外部へ自動送信しない

## OS別の見せ方

価格差を比較表で強調せず、OSを選択する2枚のカードとして表示しています

- Windows版：500円の買い切り、7日間無料試用あり
- macOS版：800円の買い切り、無料試用なし
