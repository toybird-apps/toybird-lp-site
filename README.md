# Toybird LP Site

公開用LP一式です

```text
toybird-lp-site/
├── index.html
├── robots.txt
├── sitemap.xml
├── GA4_SETUP_NOTES.md
├── ai-memorize-sheet/
├── pocket-screen/
└── prompt-ready/
    ├── index.html
    ├── styles.css
    ├── script.js
    ├── favicon.svg
    ├── UPDATE_NOTES.md
    └── assets/
        ├── app-icon.png
        ├── app-store-badge.svg
        └── og-image.png
```

## 公開URL

- AI暗記シート：`https://lp.toybird.com/ai-memorize-sheet/`
- Pocket Screen：`https://lp.toybird.com/pocket-screen/`
- Prompt Ready：`https://lp.toybird.com/prompt-ready/`

## 共通設定

- GA4測定ID：`G-MTD9Z8S7QG`
- ルートの `index.html` は空白かつnoindex
- 公開対象LPは `sitemap.xml` に登録
- `robots.txt` からサイトマップを案内
