FAQ

Frequently asked questions

Prompt Ready is not an AI model. It helps organize the request before you send it to an AI service

Can Prompt Ready provide an AI answer by itself?

No
Create a request in Prompt Ready, review it, and paste it into ChatGPT, Gemini, Claude, or another AI service

Are the Windows and macOS versions different?

Core features are shared, including request creation, use ideas, and restoring recent requests
Some interface and launch behavior differs to match each operating system

Do I need a paid AI plan?

Prompt Ready does not require a specific AI service or plan
Use the generated request according to the terms of the AI service you choose

Is my input saved?

Up to 20 recently copied requests are stored locally and can be deleted individually or all at once
Nothing is automatically sent to an external server
